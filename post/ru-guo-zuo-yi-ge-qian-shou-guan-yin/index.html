前言

这篇教程并不是教大家做黑产，作为一个普通的消费者要把眼光放在合理合法的范畴内。不违法，不打擦边球是要坚持的原则！

千手观音，佛系薅毛～

1. 一些背景

背景是灰色的。如果你对颜色不感兴趣可以跳过第一部分，直接看第二部分以后的实际操作。

很多人可能都用过小号，手机黑卡就是这样的手机小号。大多数人使用小号只是想避免不必要的骚扰，而黑产使用手机黑卡，却是为了薅羊毛，甚至干一些违法的事情。

1.1 什么是猫池

为了利益最大化，手机黑卡往往被一种叫“猫池”的设备批量养护使用，通过这种设备手机黑卡即可连上通信网络，收发短信，且全程完全自动化，在秒杀活动中，速度远快于正常用户。

1.2 什么是卡商

通讯营业厅代理商是手机黑卡的一大来源。一方面是运营商在激烈的竞争环境下，面临着巨大的销售新卡的压力，另一方面黑产对于手机卡有着巨大的需求量，两者很容易就一拍即合。

可能有同学会问，大批量的开卡，如何解决实名问题呢？

黑市上有一种工具叫“虚拟二代身份证阅读器”，这种工具模拟真实的身份证阅读器，可将预定的身份证信息提交到运营商的实名系统，实现批量实名注册。

随着国内手机卡实名要求越来越严，打击力度不断加大，卡商也在寻求另一种有效的途径：出海，可以说卡商国际化的脚步比某些互联网公司要快不少。

1.3 什么是接码平台

随着网络安全法的实施，监管对网络服务商提出了更高的实名要求，各类网络服务提供商对用户的验证，都基于短信验证码方案来实现。黑产要突破这一防线，需要大量的手机卡来接收验证码，有需求就有市场，黑市上也就出现前面说的接码平台。

有了接码平台，黑产人员就不需要自己去养手机卡了，通过接码平台的获取手机号 API 接口，就可以拿到别人猫池中养的手机号，黑产人员将手机号提交给各类网络服务商，再调用接码平台的API接口，获取验证码内容即可完成。

游戏和电商是互联网离钱最近的两个场景，自然也是黑产关注最多的两个场景。在游戏场景下，除了游戏帐号，还有游戏中的装备、游戏货币；而在电商场景下，各类薅羊毛活动更是数不胜数。

而与前几年的情况不同的是，最近越来越多的黑产人员利用接码平台的手机黑卡来炒虚拟币，比特币的造富故事吸引了国内很多创业团队开始了「造币」运动，各类虚拟币应接不暇。这些虚拟币的交易平台为了吸引用户，大都发起了各式各样的拉新活动，比如邀请注册成功后送币，黑产人员当然不会放过这「潜在暴富」的机会，利用接码平台注册帐号，用来屯币，一旦虚拟币涨价后就可以套现了。

2. 怎么实现一卡多号

大家都知道目前国内主流的运营商有：移动、联通和电信，其实这三家运营商都给我们提供了正规的小号解决方案。

当然阿里也给大家提供了小号方案。

那我们挑一种 —— 移动的和多号来说说，篇幅有限，其他的运营商大家可以自己尝试。

2.1 注册小号

下载和多号，点击下方“靓号超时福利季”的 banner，进行选号，支持4位尾号个性定制，还是不错的。

一个小号的成本是5元/月，12元/季、21元/半年、36元/年，费用由主号扣除，再赠你几天爱奇艺会员。附功能介绍

选好号以后，开始进行实名认证。

⚠️ 国家规定，每个人全网全国最多5张实名通讯卡，三大运营商都包括在内，和多号一个主卡最多拥有3个小号，且小号是算在实名通讯卡里的。

举个 🌰 ，你有一张移动的实名卡，然后又在和多号上申请了3个小号，那么你在全网全国最多只能再开一张实名卡。

实名认证成功后会收到办理成功的短信。但这并不意味着现在就可以使用小号。

要等待审核，审核通过后就可以使用了。

一般几个小时就审核完了，据说去移动营业厅申请的话貌似审核会快很多。

小号可以做一些个性化的设置。

更多细节个规则可以查询和多号帮助中心

2.2 阿里小号

这里特别说一下阿里小号，因为它...不需要实名认证（当然它是通过淘宝、支付宝账户登录的）。最多支持两个小号。有效期有三档：5元/月，12元/半年，20元/年。注册过程这里就不赘述了。

相比运营商的特点是可换号！成本3元/次。不过每天数量有限。

3. 怎么实现微信影分身

多重影分身？？！难道是这样......

答案当然不是！，上面讲的这些小号无论是不是实名都不能直接注册微信号！！！

所以正确的结印是什么样的呢？

3.1 注册微信小号

来说一下目前配置：

  • 一个实名手机号（不是实名小号）
  • 一个绑定了手机号，并且绑定了QQ或者邮箱的微信号（缺一不可）
  • 手上有多余的手机或者直接在本机再注册一个微信号也可以

这些配置基本是一个正常的微信号所该有的安全设置，大家应该都能满足。

按照目前微信的安全规则，一个微信号在绑定了QQ或者邮箱的基础上是可以没有绑定的手机号的！但是我们无法直接解绑手机号，它只能换绑。

所以我们可以直接用现在的这个手机号重新注册微信号
